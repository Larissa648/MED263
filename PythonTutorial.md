# MED 263 Python Tutorial

## Data types

#### Numbers
```python
>>> 1 + 2
3
>>> 5 + 5 + 5
15
>>> 5 ** 3
125
>>> 5^3
6
>>> help('^')
>>> 1^1
0
>>> 1^0
1
>>> 12 / 5
2
>>> 11 / 5
2
>>> 11.0 / 5
2.2
```

#### String
```python
>>> motif = 'CCTTATAAGG'
>>> len(motif)
10
>>> motif[0]
'C'
>>> motif[1]
'C'
>>> motif[2]
'T'
>>> motif[0:3]
'CCT'
>>> motif[4:7]
'ATA'
>>> motif[-2]
'G'
```
